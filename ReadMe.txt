**** 启动游戏 ***********************************

	运行文件夹中的SCP - Containment Breach.exe
	选择适合你屏幕的分辨率并点击“启动”按钮

	SCP - 收容失效使用Blitz3D引擎，但该引擎已过于古老且过时
	在你使用较新的操作系统运行游戏时，你可能会遇到一些问题

	如果你启动游戏时遇到"Memory Access Violation"（内存访问冲突）的问题或者
	"Unable to set graphics mode"（无法设置显示模式），请尝试以下方式：

	- 以管理员身份运行
	- 检查你的显卡驱动更新
	- 尝试不同的分辨率或全屏运行
	- 尝试使用不同的兼容模式运行


**** 默认键位 ***********************************

	WASD			-	移动
	空格			-	手动眨眼
	左Shift			-	疾跑
	左Control			-	蹲下
	Tab			-	物品栏
	F5			-	保存
	F3			-	[数据删除]

	按左键来捡起物品或使用拉杆、按钮

	双击物品栏的物品来使用

	将物品从物品栏拖拽出来以丢弃物品

	将一个物品拖拽到其他物品上以组合某些物品
